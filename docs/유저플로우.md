# 유저플로우 — 대진대 택시 타자

> 사용자가 앱에 진입하여 회원가입을 거치고, 방을 개설하거나 신청하여 함께 탑승하고 정산하기까지의 화면 흐름을 정의합니다.

## 1. 주요 사용자 흐름

### A. 방장 흐름 (Host Flow)
```text
로그인/회원가입 
  ➔ 홈 화면 (우측 하단 [+] 플로팅 버튼 클릭)
  ➔ 방 만들기 화면 (출발/목적지, 출발시간, 모집인원, 성별 필터, 정산 계좌 입력 후 개설)
  ➔ 신청자 관리 화면 (신청 대기자 목록 확인 ➔ [수락] 또는 [거절] 선택)
  ➔ 실시간 채팅방 (채팅 소통, 정산 공지 고정 확인)
  ➔ 여정 진행에 따른 상태 변경 ([모집 마감] ➔ [탑승 중] ➔ [정산 요청] 클릭)
  ➔ [정산 요청] 단계 시 '실제 총 택시 요금' 입력 ➔ 참여자에게 N/1 정산금 자동 고지
```

### B. 참여자 흐름 (Guest Flow)
```text
로그인/회원가입 
  ➔ 홈 화면 (나의 등하교 경로 입력 ➔ 추천 팟 및 전체 리스트 조회)
  ➔ 모집 카드 상세 정보 확인 및 [참여 신청] 버튼 클릭
  ➔ 방장 수락 대기 (신청 상태: 대기 중)
  ➔ 방장 수락 완료 시 ➔ 실시간 채팅방 활성화 및 자동 입장
  ➔ 실시간 채팅방 (채팅 소통, 상단 고정 공지 확인)
  ➔ 방장이 [정산 요청]으로 상태 변경 시 ➔ 요금 고지 배너 확인 ➔ 송금 완료 처리
```

## 2. 화면 목록

| 화면 | 하는 일 | 로그인 필요? | 다음 이동 |
| --- | --- | --- | --- |
| **로그인 / 회원가입** | 이메일/비밀번호 인증, 학번 및 성별 정보 입력 | 아니오 | 성공 시 홈 화면 |
| **매칭 홈 (Home)** | 등하교 경로 필터 입력, 추천 팟 조회, 전체 모집 리스트 조회, 방 만들기 이동 | 예 | 방 만들기, 방 상세/신청자 관리(방장), 채팅방(수락자) |
| **방 만들기 (Create)** | 출발지, 목적지, 출발시간, 모집인원, 성별 필터, 계좌번호 입력 및 방 생성 | 예 | 홈 화면 (생성 완료 후) |
| **신청자 관리 (Manage)** | (방장 전용) 신청 대기 인원 프로필(학번, 성별) 확인 및 수락/거절 처리 | 예 | 채팅방 (대화 시작) |
| **실시간 채팅방 (Chat)** | (수락된 인원 전용) 실시간 대화, 탑승 상태 변경 버튼(방장), 정산 공지 배너 및 N/1 계산기 노출 | 예 | 홈 화면 (뒤로가기) |

## 3. 핵심 시나리오

### 시나리오 A. 처음 온 학생이 회원가입 후 택시 팟에 참여하여 정산까지 마친다
1. 앱에 처음 접속하면 **로그인/회원가입** 화면이 뜬다.
2. [회원가입] 탭을 눌러 이메일, 비밀번호, 학번, 성별을 입력하고 가입한다.
3. 자동으로 로그인되어 **매칭 홈**으로 이동한다.
4. 홈 상단에서 출발지 `대진대역`, 목적지 `대진대 정문`을 선택한다.
5. 추천 팟 및 아래 목록에서 나에게 맞는 시간과 성별 조건의 방을 고르고 **[참여 신청]**을 누른다.
6. 방장이 신청을 **[수락]**하면, 해당 방 카드가 **[채팅방 입장]**으로 바뀌며 채팅방에 접속한다.
7. 채팅방에서 동승자들과 만날 위치를 정하고 택시에 탑승한다.
8. 운행 종료 후 방장이 **[정산 요청]** 상태로 변경하고 총 요금을 입력하면, 채팅창에 "인당 1,800원 송금" 가이드 배너가 뜬다.
9. 방장의 계좌로 송금하고 여정을 마친다.

## 4. 예외 흐름

- **성별 불일치 신청 제한**: 성별 필터가 '동성끼리만'으로 설정된 방에 성별이 다른 참여자가 신청하려고 할 때 경고 팝업을 띄우고 신청 버튼을 비활성화한다.
- **인원 초과 제한**: 이미 정원이 가득 찬 방에 대해서는 추가 수락을 차단하고, 자동으로 상태를 '모집 마감'으로 변경한다.
- **채팅 비권한자 차단**: 해당 방의 방장이거나 수락된(accepted) 참여자가 아닌 유저가 채팅방 URL로 직접 접근을 시도하면 홈 화면으로 리다이렉트한다.
- **네트워크 오류**: Supabase Realtime 연결이 일시적으로 해제된 경우 상단에 '실시간 연결 끊김. 재연결 시도 중...' 토스트를 노출한다.
